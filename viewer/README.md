# Session Log Viewer

Local browser-based viewer for the triage agent's JSONL session logs.

Open `index.html` in a browser, then pick or drop a `.jsonl` file from
the `logs/` directory.

See [docs/session-log-viewer.md](../docs/session-log-viewer.md) for the
full reference: panes, compare view, security model, and the coupling
to the triage agent's architecture.
